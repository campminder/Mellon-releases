# Mellon

macOS dictation app powered by OpenAI. Speak, friend, and enter.

## Install

1. **[Download the latest release](https://github.com/campminder/Mellon-releases/releases/latest/download/Mellon-0.2.0.zip)**
2. Unzip and drag **Mellon.app** to `/Applications`
3. Open Mellon — macOS will block it the first time:
   - Click **Done** on the warning dialog
   - Go to **System Settings → Privacy & Security**
   - Scroll down and click **Open Anyway** next to the Mellon message
   - This is a one-time step
4. Grant **Microphone** and **Accessibility** permissions when prompted
5. Click the mic icon in your menu bar → **Settings** → enter your OpenAI API key

## Usage

- **Double-tap Control** to start dictating, **tap Control once** to stop
- Or set a custom shortcut in Settings
- Text is transcribed and pasted into whatever app has focus

## Requirements

- macOS 14+
- OpenAI API key ([get one here](https://platform.openai.com/api-keys))

## Updates

Mellon checks for updates automatically on launch. You can also check manually from the menu bar → **Check for Updates...**
