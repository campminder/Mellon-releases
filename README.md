# Mellon

macOS dictation app powered by OpenAI. Speak, friend, and enter.

## Install

1. **[Download the latest release](https://github.com/campminder/Mellon-releases/releases/latest)**
2. Click the `.zip` file to download
3. Unzip and drag **Mellon.app** to `/Applications`
4. Open Mellon (right-click → Open the first time to bypass Gatekeeper)
5. Grant Microphone and Accessibility permissions when prompted
6. Click the mic icon in your menu bar → **Settings** → enter your OpenAI API key

## Usage

- **Double-tap Control** to start dictating, **tap Control once** to stop
- Or set a custom shortcut in Settings
- Text is transcribed and pasted into whatever app has focus

## Requirements

- macOS 14+
- OpenAI API key ([get one here](https://platform.openai.com/api-keys))

## Updates

Mellon checks for updates automatically on launch. You can also check manually from the menu bar → **Check for Updates...**
